WEB Developer

Apa itu web developer ??

Web developer adalah seorang programmer yang membuat program dan aplikasi untuk world wide web. Lalu apa saja yang dipelajari oleh web developer ? Seorang web developer mempelajari Front-end dan Back-end untuk mengembangkan sebuah website. lebih lengkapnya mari kita pelajari bersama.

Yuk Pelajari !!

Apa saja yang dipelajari ?

Front-end

Front-End adalah sebuah bagian dari website yang menyuguhkan tampilan kepada user. Bagian ini dibangun menggunakan HTML, CSS, dan JavaScript

HTML

HTML memungkinkan user untuk membuat dan menyusun bagian paragraf, heading, link, dan blockquote untuk halaman web dan aplikasi.

CSS

CSS biasanya digunakan untuk mengatur tampilan elemen yang tertulis dalam bahasa markup

JavaScript

Javascript merupakan salah satu bahasa pemrograman web yang bisa membuat halaman website anda lebih menarik serta lebih hidup.

Back-end

Backend adalah bagian pada website untuk mengatur logika aplikasi dan mengolah data-datanya nanti

PHP

PHP (Hypertext Preprocessor) adalah sebuah bahasa pemrograman server side scripting yang bersifat open source.

Python

Python adalah bahasa pemrograman tujuan umum yang ditafsirkan, tingkat tinggi. Dibuat oleh Guido van Rossum dan pertama kali dirilis pada tahun 1991.

SQL

SQL (Structured Query Language) adalah sebuah bahasa yang digunakan untuk mengakses data dalam basis data relasional.
